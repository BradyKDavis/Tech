Found in the ProgramExercises folder is the eclipse project files used
to create this java program. Screenshots included are generated from the
included JLineChart program. This graphing utility uses the JFreeChart
library to display data generated by the program. Also included is a
JUnit test case used to ensure the program is working as intended.

The database schema can be found in the included test-_2015-11-11.sql file. The database 
for the created table is "test" and the included sql also includes the data 
tested on my local mysql instance.
